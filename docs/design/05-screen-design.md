## 2. ルーティング（expo-router）

```
app/
├── _layout.tsx              # Tabs ルートレイアウト
├── index.tsx                # ホーム（一覧）
├── chart.tsx                # グラフ
├── settings.tsx             # 設定
└── record/
    ├── _layout.tsx          # Stack レイアウト
    ├── [date].tsx           # 入力/編集（記録日パラメタ）
    └── detail/[date].tsx    # 詳細
```

`record/*` はモーダル風 push（戻る矢印で戻れる）。

## 3. 各画面ワイヤーフレーム

### 3.1 ホーム / 一覧画面 (`index.tsx`)

```
┌──────────────────────────────────┐
│ 気分・睡眠ログ           ⚙        │  ← ヘッダ
├──────────────────────────────────┤
│ [カレンダー] [リスト]             │  ← 表示切替タブ
├──────────────────────────────────┤
│        2026年 4月          ◀ ▶  │
│ ┌──┬──┬──┬──┬──┬──┬──┐         │
│ │日│月│火│水│木│金│土│         │
│ ├──┼──┼──┼──┼──┼──┼──┤         │
│ │   │   │   │ 1 │ 2 │ 3 │ 4 │ │
│ │   │   │   │🙂+1│😐 0│😢-2│🙂+1│ │
│ │   │   │   │7h │6h │5h │8h │ │
│ ├──┼──┼──┼──┼──┼──┼──┤         │
│ │ 5 │ 6 │ 7 │ 8 │ 9 │10 │11 │ │
│ │😄+2│🙂+1│   │   │   │   │   │ │
│ │8h │7h │   │   │   │   │   │ │
│ └──┴──┴──┴──┴──┴──┴──┘         │
│                                  │
│              ┌────┐              │
│              │ +  │  ← 記録FAB   │
│              └────┘              │
├──────────────────────────────────┤
│  🏠ホーム   📊グラフ   ⚙設定     │
└──────────────────────────────────┘
```

**動作:**
- カレンダーセル: **気分絵文字 + スコア数値（横並び）** + 睡眠合計時間（記録あり）/ 空（記録なし）
- セルタップ: 記録ありなら詳細画面、なしなら入力画面（その日付で）
- FAB: 昨日の入力画面へ
- ヘッダー右の歯車: 設定タブへ
- リストタブ（**v1.1〜v1.3**: 日付＋絵文字＋スコア / タグ / イベント / メモ の最大 4 行）:
  ```
  2026/4/30(火)       🙂 +1     ← v1.3: 通常行に西暦年付与
  楽しい、感謝
  📅 梅田でショッピング         ← v1.3: イベントがある日のみ
  友人と食事

  今日 4/29(月)       😢 -1     ← 「今日 / 昨日」プレフィックスがある行は年なし
  不安、鬱
  眠りが浅かった

  2026/4/28(日)       😐  0
  ```
  - スコアは `+1 / 0 / -2` のように符号付き
  - タグが空なら表示しない、イベントが空なら表示しない、メモが空なら表示しない（行省略）
  - **検索ハイライト（v1.1〜v1.3）**: キーワード検索時、メモ・イベント内のマッチ部分を赤太字で強調。タグ絞り込み時、選択されているタグ名を赤太字で強調
  - **検索対象（v1.3）**: メモ / 感情タグ / イベント の OR（大文字小文字区別なし）
  - **西暦年表示（v1.3）**: 通常行は `YYYY/M/D(曜)`、今日/昨日プレフィックスのある行は省略

```
┌──────────────────────────────────┐
│ ← 4月30日(火)              保存  │  ← ヘッダ
├──────────────────────────────────┤
│ 📅 記録日                        │
│   [ 2026年4月30日(火)        ▼ ] │
├──────────────────────────────────┤
│ 😊 気分                          │
│   ┌────────────────────────────┐ │
│   │ 😢   🙁   😐   🙂   😄    │ │
│   │ -2   -1    0   +1   +2    │ │
│   │              ●            │ │  ← 選択中
│   └────────────────────────────┘ │
├──────────────────────────────────┤
│ 🏷 感情タグ                       │
│  [ネガティブ感情] ▼              │
│   不安  怒り  ●イライラ  ...    │
│  [ポジティブ感情] ▼              │
│   ●楽しい  嬉しい  穏やか ...   │
│  [状態] ▼                        │
│   鬱  メンタルダウン  ...        │
│  [行動・思考] ▼                  │
│   「自分は無能だ」 ...           │
├──────────────────────────────────┤
│ 🛏 睡眠時間帯  合計 7h20m        │
│   21  23  01  03  05  07  09 11 │
│   ├──┼──┼──┼──┼──┼──┼──┤      │
│       ▓▓▓▓▓▓▓                   │  ← ドラッグ可
│                ▓▓▓▓▓▓▓▓         │
│  [+ 区間を追加]                  │
├──────────────────────────────────┤
│ 📅 イベント（任意）（v1.3）       │
│   ┌────────────────────────────┐ │
│   │ 梅田でショッピング         │ │  ← 単一行 / 200 文字上限
│   └────────────────────────────┘ │
├──────────────────────────────────┤
│ 📝 メモ（任意）                  │
│   ┌────────────────────────────┐ │
│   │                            │ │
│   └────────────────────────────┘ │
├──────────────────────────────────┤
│ 💊 睡眠導入剤（v1.2）            │
│   ⦿ なし                         │
│   ◯ ルネスタ 0.25mg〜3.0mg (8段) │
├──────────────────────────────────┤
│ 💊 頓服薬（v1.2）                │
│   ⦿ なし                         │
│   ◯ ルネスタ 1.0/2.0/3.0mg       │
└──────────────────────────────────┘
```

**動作:**
- 「保存」: zod バリデーション → DB 保存 → ホームへ戻る
- 戻る: 確認なしで前画面に戻る（v1.0 にあった「下書き保持／破棄」モーダルは v1.1 で撤去）
- 感情タグ：4カテゴリのアコーディオン、複数選択可
- **服薬記録（v1.2）**: 単一選択ラジオ、デフォルト「なし」。詳細は [17-v1.2-medications.md](17-v1.2-medications.md)
- **イベント（v1.3）**: 単一行 TextInput、200 文字上限、メモ欄の直前に配置。詳細は [18-v1.3-event-field.md](18-v1.3-event-field.md)

> **v1.1 撤去**: 自動下書き保存と「下書きを復元しますか？」モーダルは UX を阻害するため撤去（[16-v1.1-features.md §6](16-v1.1-features.md)）。

## 4. 共通モーダル/ダイアログ

| 名前 | トリガ | 内容 |
|------|--------|------|
| 削除確認 | 詳細画面の削除ボタン | 「この記録を削除しますか？」/ キャンセル / 削除 |
| 区間上限 | 11個目追加時 | トースト「最大10区間まで登録できます」 |
| インポート確認（2段階） | 設定の CSV インポート | 警告 → 最終確認の 2 ダイアログ |

> v1.0 の「下書き復元」「未保存破棄」モーダルは v1.1 で撤去。

## 5. デザインシステム

| 要素 | 値 |
|------|-----|
| カラー（気分） | 緑 #4CAF50 |
| カラー（睡眠時間） | 青 #1E88E5 |
| カラー（睡眠時間帯） | 赤 #E53935 |
| 背景 | システム標準（ダーク/ライト両対応） |
| フォント | システム標準（San Francisco / Roboto） |
| 角丸 | 8pt |
| タップ最小 | 44pt 四方 |
| ジェスチャー触覚 | `expo-haptics` の `selectionAsync` |

## 6. アクセシビリティ

- 色だけに頼らない（折れ線にはマーカーも併用、タグは色＋文字）
- VoiceOver/TalkBack 対応（全要素に `accessibilityLabel`）
- ダイナミックタイプ対応（フォントサイズはユーザー設定追従）
- カラーコントラスト WCAG AA 以上

## 7. 状態とナビゲーションの相互作用

| 操作 | 結果 |
|------|------|
| 入力画面で保存 | DailyRecord upsert + ホームへ戻る + 一覧クエリ再取得 |
| 入力画面で戻る | そのまま戻る（v1.1 で確認モーダル廃止） |
| 入力画面で戻る（保存済） | そのまま戻る |
| カレンダーで未記録日タップ | `record/[date]` を push |
| 一覧で記録ありセルタップ | `record/detail/[date]` を push |
| グラフでタブ切替 | 同画面内で再描画、navigation スタックは変えない |

## 8. オープン課題（実装段階で確定）
