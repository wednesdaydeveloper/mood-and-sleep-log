## 2. ルーティング（expo-router）

```
app/
├── _layout.tsx              # Tabs ルートレイアウト
├── index.tsx                # ホーム（一覧）
├── chart.tsx                # グラフ
├── settings.tsx             # 設定
└── record/
    ├── _layout.tsx          # Stack レイアウト
    ├── [date].tsx           # 入力/編集（記録日パラメタ）
    └── detail/[date].tsx    # 詳細
```

`record/*` はモーダル風 push（戻る矢印で戻れる）。

## 3. 各画面ワイヤーフレーム

### 3.1 ホーム / 一覧画面 (`index.tsx`)

```
┌──────────────────────────────────┐
│ 気分・睡眠ログ           ⚙        │  ← ヘッダ
├──────────────────────────────────┤
│ [カレンダー] [リスト]             │  ← 表示切替タブ
├──────────────────────────────────┤
│        2026年 4月          ◀ ▶  │
│ ┌──┬──┬──┬──┬──┬──┬──┐         │
│ │日│月│火│水│木│金│土│         │
│ ├──┼──┼──┼──┼──┼──┼──┤         │
│ │  │  │  │ 1│ 2│ 3│ 4│         │
│ │  │  │  │🙂│😐│😢│🙂│         │
│ │  │  │  │7h│6h│5h│8h│         │
│ ├──┼──┼──┼──┼──┼──┼──┤         │
│ │ 5│ 6│ 7│ 8│ 9│10│11│         │
│ │😄│🙂│  │  │  │  │  │         │
│ │8h│7h│  │  │  │  │  │         │
│ └──┴──┴──┴──┴──┴──┴──┘         │
│                                  │
│              ┌────┐              │
│              │ +  │  ← 記録FAB   │
│              └────┘              │
├──────────────────────────────────┤
│  🏠ホーム   📊グラフ   ⚙設定     │
└──────────────────────────────────┘
```

**動作:**
- カレンダーセル: 気分絵文字 + 睡眠合計時間（記録あり）/ 空（記録なし）
- セルタップ: 記録ありなら詳細画面、なしなら入力画面（その日付で）
- FAB: 昨日の入力画面へ
- ヘッダー右の歯車: 設定タブへ
- リストタブ:
  ```
  📅 4/30(火)  🙂  7h20m  [楽しい, 感謝]
  📅 4/29(月)  😢  6h10m  [不安, 鬱]
  📅 4/28(日)  😐  7h00m  ―
  ...
  ```

### 3.2 記録入力画面 (`record/[date].tsx`)

```
┌──────────────────────────────────┐
│ ← 4月30日(火)              保存  │  ← ヘッダ
├──────────────────────────────────┤
│ 📅 記録日                        │
│   [ 2026年4月30日(火)        ▼ ] │
├──────────────────────────────────┤
│ 😊 気分                          │
│   ┌────────────────────────────┐ │
│   │ 😢   🙁   😐   🙂   😄    │ │
│   │ -2   -1    0   +1   +2    │ │
│   │              ●            │ │  ← 選択中
│   └────────────────────────────┘ │
├──────────────────────────────────┤
│ 🏷 感情タグ                       │
│  [ネガティブ感情] ▼              │
│   不安  怒り  ●イライラ  ...    │
│  [ポジティブ感情] ▼              │
│   ●楽しい  嬉しい  穏やか ...   │
│  [状態] ▼                        │
│   鬱  メンタルダウン  ...        │
│  [行動・思考] ▼                  │
│   「自分は無能だ」 ...           │
├──────────────────────────────────┤
│ 🛏 睡眠時間帯  合計 7h20m        │
│   21  23  01  03  05  07  09 11 │
│   ├──┼──┼──┼──┼──┼──┼──┤      │
│       ▓▓▓▓▓▓▓                   │  ← ドラッグ可
│                ▓▓▓▓▓▓▓▓         │
│  [+ 区間を追加]                  │
├──────────────────────────────────┤
│ 📝 メモ（任意）                  │
│   ┌────────────────────────────┐ │
│   │                            │ │
│   └────────────────────────────┘ │
└──────────────────────────────────┘
```

**動作:**
- 値変更ごとに自動下書き保存（debounce 500ms）
- 「保存」: zod バリデーション → DB 保存 → ホームへ戻る
- 戻る: 未保存変更があれば「下書き保持して戻る／破棄」モーダル
- 入力時に下書きがあれば「下書きを復元しますか？」モーダルを最初に表示
- 感情タグ：4カテゴリのアコーディオン、複数選択可

## 4. 共通モーダル/ダイアログ

| 名前 | トリガ | 内容 |
|------|--------|------|
| 削除確認 | 詳細画面の削除ボタン | 「この記録を削除しますか？」/ キャンセル / 削除 |
| 下書き復元 | 入力画面マウント時 | 「未保存の下書きがあります。復元しますか？」/ 破棄 / 復元 |
| 未保存破棄 | 入力画面で戻る時 | 「変更が保存されていません」/ 戻る / 破棄して戻る |
| 区間上限 | 11個目追加時 | トースト「最大10区間まで登録できます」 |

## 5. デザインシステム

| 要素 | 値 |
|------|-----|
| カラー（気分） | 緑 #4CAF50 |
| カラー（睡眠時間） | 青 #1E88E5 |
| カラー（睡眠時間帯） | 赤 #E53935 |
| 背景 | システム標準（ダーク/ライト両対応） |
| フォント | システム標準（San Francisco / Roboto） |
| 角丸 | 8pt |
| タップ最小 | 44pt 四方 |
| ジェスチャー触覚 | `expo-haptics` の `selectionAsync` |

## 6. アクセシビリティ

- 色だけに頼らない（折れ線にはマーカーも併用、タグは色＋文字）
- VoiceOver/TalkBack 対応（全要素に `accessibilityLabel`）
- ダイナミックタイプ対応（フォントサイズはユーザー設定追従）
- カラーコントラスト WCAG AA 以上

## 7. 状態とナビゲーションの相互作用

| 操作 | 結果 |
|------|------|
| 入力画面で保存 | DailyRecord upsert + draftRecord 削除 + ホームへ戻る + 一覧クエリ再取得 |
| 入力画面で戻る（未保存） | 確認モーダル → 破棄なら draftRecord 削除 |
| 入力画面で戻る（保存済） | そのまま戻る |
| カレンダーで未記録日タップ | `record/[date]` を push（draft があれば復元提案） |
| 一覧で記録ありセルタップ | `record/detail/[date]` を push |
| グラフでタブ切替 | 同画面内で再描画、navigation スタックは変えない |

## 8. オープン課題（実装段階で確定）

- カレンダー時の月送りジェスチャー（左右スワイプの追加）
- 縦帯グラフのタップ判定領域（帯の境界＋ヒットマージン）
- ダーク/ライトテーマの色微調整
