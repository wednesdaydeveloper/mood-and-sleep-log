# チュートリアル・オンボーディング

## 1. スコープ

自分専用アプリだが、**作った直後は自分も使い方を忘れる**。最低限のオンボーディングは入れる。

## 2. 初回起動フロー

```
アプリ起動
  ↓
初回フラグ判定（AsyncStorage の `hasOnboarded`）
  ↓
未済 → ウェルカム画面
  ↓
3〜4ステップのスライドで使い方説明
  ↓
完了 → ホーム画面
```

## 4. 実装

### ルーティング

```
app/
├── _layout.tsx
├── (onboarding)/
│   ├── _layout.tsx        # スワイプ可能な Stack または Pager
│   ├── welcome.tsx
│   ├── morning.tsx
│   ├── chart.tsx
│   └── ready.tsx
├── (tabs)/
│   ├── _layout.tsx
│   ├── index.tsx
│   ├── chart.tsx
│   └── settings.tsx
```

`_layout.tsx` で `hasOnboarded` を判定し、未済なら `(onboarding)` へリダイレクト。

### 状態管理

```typescript
// src/lib/onboarding.ts
import AsyncStorage from "@react-native-async-storage/async-storage";

export async function hasCompletedOnboarding(): Promise<boolean> {
  return (await AsyncStorage.getItem("hasOnboarded")) === "true";
}

export async function completeOnboarding(): Promise<void> {
  await AsyncStorage.setItem("hasOnboarded", "true");
}
```

「はじめる」タップで `completeOnboarding()` → `router.replace("/")`。

## 5. 機能内ヒント（チュートリアルではなく）

オンボーディング後も、初めての画面では小さなヒントを表示:

| 画面 | 初回ヒント |
|------|-----------|
| 入力画面 | 「タイムラインをドラッグして睡眠時間を選択」（吹き出し、3秒で消える） |
| グラフ画面 | 「タブを切り替えて期間変更」（同上） |
| カレンダー | 「日付タップで詳細表示」 |

実装: `seenHints` を AsyncStorage に保持、画面ごとに1度だけ表示。

## 7. アクセシビリティ

- スワイプだけでなく `[次へ]` ボタンも提供（VoiceOver/TalkBack で操作可能）
- 各画面にスキップボタン（右上に「スキップ」）
- 画像とテキストの両方で意味を伝える

## 8. 実装タイミング

- **M7（仕上げ）** で実装
- 機能実装完了後、最後にユーザー体験を整える形

## 9. テスト

- E2E: 初回起動 → オンボーディング完走 → ホーム表示
- AsyncStorage を `clearState: true` でリセットしてテスト

## 10. デザイン

- スライドはフルスクリーン
- 中央に大きな絵文字、その下にタイトル + 説明文
- 下部にページインジケータ（● ● ○ ●）
- 配色は §08 デザインシステムに準拠（背景: `bg.primary`、強調: `accent`）
