# 気分・睡眠記録アプリ 要件定義書

## 0. メモ

- 添付ファイル（手書きグラフモックアップ `IMG_0395.JPG`）を反映済み。
- 本ドキュメントは要件定義のみ。実装計画・設計は要件確定後に別途作成します。

---

## 1. プロジェクト概要

### 1.1 目的
日々の **気分・睡眠時間帯・睡眠時間** を簡単に記録し、過去の傾向を可視化することで、自身のメンタル/睡眠状態の把握を支援する自分専用アプリ。

### 1.2 想定利用者
- シングルユーザー（アプリ作成者本人）
- 認証機能なし

### 1.3 利用シーン
- 朝起きた直後に、前日（昨日）の気分と睡眠を記録する
- 任意のタイミングで過去の記録を振り返る、グラフで傾向を確認する

---

## 2. 技術前提

| 項目 | 内容 |
|------|------|
| プラットフォーム | スマホネイティブ（React Native / Expo を想定） |
| 対応OS | iOS / Android 両方 |
| データ保存 | 端末ローカルDB（SQLite を推奨） |
| 認証 | なし |
| オフライン | 完全オフライン動作（ネット接続不要） |
| バックアップ | CSV エクスポート機能あり（§4.4） |

---

## 3. 用語定義

| 用語 | 定義 |
|------|------|
| 記録日 | その記録が対象とする日付（例：5/1朝に記録 → 記録日は 4/30） |
| 睡眠区間 | 1つの連続した睡眠の開始時刻〜終了時刻 |
| 睡眠時間（合計） | 1記録日に紐づく全睡眠区間の時間合計 |
| 気分スケール | −2（とても悪い）/ −1 / 0（普通）/ +1 / +2（とても良い）の5段階 |
| 感情タグ | 気分を補足する複数選択ラベル（例：不安、疲れ、無力感、楽しい、感謝） |

---

## 4. 機能要件

### 4.1 記録機能（FR-1）

#### FR-1.1 日次記録の登録
- 記録日を選択（デフォルト：昨日）
- 任意の過去日付の記録も可能
- 同一記録日に対する記録は1件（再登録は上書き＝編集扱い）

#### FR-1.2 気分の入力
- 5段階スケール（−2 / −1 / 0 / +1 / +2、絵文字またはアイコン併用）必須
- 感情タグ：固定プリセットから複数選択可能（任意、ユーザー追加・編集は不可）
  - **4カテゴリに分類**して選択UI上でグルーピング表示
  - **【ネガティブ感情】(12)**: 不安 / 怒り / イライラ / 寂しさ / 無力感 / 悲しみ / 罪悪感 / 焦り / 恥ずかしい / 空虚 / 退屈 / 物足りない
  - **【ポジティブ感情】(10)**: 楽しい / 嬉しい / 穏やか / 感謝 / 達成感 / リラックス / 充実 / 希望 / 自信 / 楽
  - **【状態】(15)**: 鬱 / メンタルダウン / ストレス / オーバーワーク / 疲れ / 感情のコントロール不能 / 涙もろい / 憂鬱 / 集中できない / 軽躁 / 軽鬱 / 躁 / フラット / アイデアが駆け巡る / 過活動
  - **【行動・思考】(4)**: 「自分は無能だ」/「誰も分かってくれない」/「逃げたい」/「動けない」
  - 合計 **41 タグ**（v1.1 で 11 タグ追加）
- メモ欄（自由記述、任意）

#### FR-1.3 睡眠時間帯の入力
- **UI形式：横軸タイムライン上のドラッグ選択**
  - 横軸レンジ：**21:00 〜 翌 11:00 固定**（グラフ下段のY軸と一致）
  - ハンドルをドラッグして開始・終了を設定
  - タップで新規区間追加、長押しで削除
- **分割睡眠対応**
  - 1記録日に対し複数の睡眠区間を登録可能（例：23:00–02:00 と 03:30–07:00）
  - 区間同士の重複は不可（バリデーション）
- 区間粒度：10分単位
- 睡眠時間の合計は区間から自動計算（手入力不要）

#### FR-1.4 編集・削除
- 過去の任意の記録を編集可能
- 削除も可能（確認ダイアログあり）

#### FR-1.5 自動下書き保存（v1.1 で撤去）
- v1.0 では入力中の自動下書き保存と「下書きを復元しますか？」モーダルを実装したが、
  運用の単純化のため v1.1 で撤去した
- 「保存」ボタンを明示的に押した内容のみが永続化される
- DB スキーマ上の `draft_record` テーブルは互換性のため残存（未使用）

### 4.2 参照機能（FR-2）

#### FR-2.1 記録一覧
- **カレンダー形式とリスト形式をタブで切替可能**
- カレンダー形式：月単位グリッド、各セルに気分絵文字・睡眠合計時間を表示
- リスト形式：新しい順、1行に日付・気分・睡眠合計時間・タグ（一部）
- タップで詳細画面へ

#### FR-2.2 記録詳細
- 1記録日の全情報を表示
- 睡眠区間はタイムライン形式で可視化
- 編集・削除ボタン

### 4.3 グラフ機能（FR-3）

添付モックアップ（IMG_0395.JPG）に基づき、**X軸を共有する縦3段パネル** で構成する。

#### FR-3.1 表示単位（期間切替）
画面上部にタブ：**week / month / year**

- **week**：直近7日（X軸は日付ラベル `4/25, 4/26, …`、日次データ）
- **month**：直近1ヶ月（X軸は日付、日次データ）
- **year**：直近1年（X軸は月、**各点は月平均に集約**。下段の睡眠時間帯は月平均の就寝・起床時刻で帯を描画）
  - 記録が0件の月は **点を描画せず折れ線にギャップを作る**（前後を線で繋がない、帯も描画しない）

#### FR-3.2 3段パネルの内訳

| 段 | 種別 | データ | Y軸 | 色（参考） |
|----|------|--------|-----|-----------|
| 上 | 折れ線 | **気分** | −2 😢 / −1 🙁 / 0 😐 / +1 🙂 / +2 😄（数値+絵文字併記） | 緑 |
| 中 | 折れ線 | **睡眠時間（合計）** | 時間（例 5h〜9h、データに応じて自動） | 青 |
| 下 | 縦帯グラフ | **睡眠時間帯** | 時刻 **21:00 〜 翌 11:00 固定** | 赤 |

- 3段は同じX軸（日付）を共有
- 下段の睡眠時間帯は、各日付列に **就寝〜起床の時刻範囲を縦の帯** として描画
- **分割睡眠の場合は1つの日付列に複数の帯が縦方向に分かれて描画**（例：23:00–02:00 の帯と 03:30–07:00 の帯が間を空けて2つ並ぶ）
- 帯のY軸は日をまたぐため、**21:00 〜 翌 11:00 固定**（入力UIの横軸と一致）

#### FR-3.3 操作
- 期間切替タブ（week / month / year）
- 期間のスクロールまたは前後送り
- 任意点・任意帯のタップで該当日の値・時刻を表示

### 4.4 データエクスポート（FR-4）

- 設定画面から **CSV エクスポート** を実行
- エクスポート対象：全 DailyRecord と SleepInterval
- CSV 列：`date, moodScore, moodTags, memo, sleepIntervals`
  - 区切り文字（フィールド間）：**カンマ `,`**
  - `moodTags` 内の複数タグも **カンマ `,` 区切り**（フィールド全体は `"..."` でダブルクォート、RFC 4180 準拠）
  - `sleepIntervals` 内の複数区間も **カンマ `,` 区切り**（例：`"23:00-02:00,03:30-07:00"`）
- iOS/Android の標準シェアシート経由で保存・送信
- インポート機能は今回のスコープ外（§8）

## 5. 非機能要件

| 区分 | 要件 |
|------|------|
| 操作性 | 朝の記録は **30秒以内** で完了できること |
| パフォーマンス | グラフ表示：1秒以内（過去1年分） |
| データ容量 | 数年分の記録でも端末ローカルで動作 |
| プライバシー | 端末外への送信なし |
| 信頼性 | アプリ強制終了でも入力中データを失わない（FR-1.5 自動下書き保存） |

## 6. データモデル（暫定）

```
DailyRecord
  id: UUID
  date: DATE (UNIQUE)              -- 記録日
  moodScore: INTEGER (-2 〜 +2)    -- 気分スケール
  moodTags: TEXT[]                 -- 感情タグ（複数）
  memo: TEXT (nullable)            -- 自由記述メモ
  createdAt: TIMESTAMP
  updatedAt: TIMESTAMP

SleepInterval
  id: UUID
  recordId: FK -> DailyRecord.id
  startAt: TIMESTAMP               -- 睡眠開始時刻（10分粒度）
  endAt: TIMESTAMP                 -- 睡眠終了時刻（10分粒度）
  -- (endAt > startAt, 同一recordId内で重複禁止)

DraftRecord                        -- 自動下書き（FR-1.5）
  date: DATE (UNIQUE, PK)
  payload: JSON                    -- 入力中の全フィールド
  updatedAt: TIMESTAMP
```

## 7. 画面構成（暫定）

1. **ホーム / 記録一覧画面**：カレンダー / リスト切替タブ、記録ボタン
2. **記録入力画面**：気分入力、感情タグ、睡眠タイムライン、メモ（自動下書き保存対応）
3. **記録詳細画面**：登録内容の表示、編集/削除
4. **グラフ画面**：week/month/year 切替の3段パネル（気分・睡眠時間・睡眠時間帯）
5. **設定画面**：CSV エクスポート など
